UI Pack Toony LITE 1.2.0
------------------------------------------------------------------

	This package contains same stuffs with UI Pack: Toony PRO. The different is this package uses GUI animator FREE to demonstrate UI animations (PRO version uses GUI Animator for Unity UI).

	Features:

		• 80+ UI prefabs.
		• 300+ PSD files (UHD texures).
		• 20 demo scenes for Portrait and Lansdcape orientations.
		• 26 sample C# scripts.
		• 15 Particle that can work with Unity UI Canvas.
		• 39 sounds.
		• Support all build player platforms.
		• GUI animator FREE is included.
		
	Compatible:

		• Unity 5.5.1 or higher.

	Note:

		• GUI animator FREE shows "GUI Animator FREE" message at upper-left corner of screen for a while after scene started.
